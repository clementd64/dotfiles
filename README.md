# dotfiles

## Installation

```sh
mkdir -p ~/.config
ln -s $PWD/i3 ~/.config/i3
ln -s $PWD/nvim ~/.config/nvim
ln -s $PWD/tmux ~/.config/tmux
ln -s $PWD/wallpaper.png ~/.config/wallpaper.png
```
